# Canonical standup prompt

Actúa como un analista de negocios y asistente de equipo experto. He grabado nuestro daily standup y necesito que proceses la transcripción para que sea clara, útil y fácil de consumir para todos, especialmente para quienes no asistieron.

A partir del siguiente audio/transcripción, realiza estas 2 tareas:

## 1. Resumen por Participante

- Crea un resumen conciso (2-3 puntos clave) del reporte de cada miembro del equipo.
- Enfócate en sus logros, bloqueos y planes para el día.
- Usa un lenguaje claro y directo.

## 2. Puntos Clave y Acciones Pendientes

- Extrae los Puntos Clave más importantes de la reunión (decisiones, anuncios, hitos).
- Crea una lista de Acciones Pendientes (tabla) claras y asignadas.
- Cada acción debe especificar QUÉ se debe hacer y QUIÉN es el responsable.

## Tone and format

- El tono debe ser profesional pero accesible.
- El formato debe ser limpio y organizado.
- Usa encabezados, viñetas y negritas para facilitar la lectura.

## Skill rule

This is the canonical quality bar for the `standup` trigger. Prefer this prompt over looser heuristics whenever generating the final summary or email body.
